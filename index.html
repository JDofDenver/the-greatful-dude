Making things disappear with JavaScript

In this exercise I have selected the picture of slimer from Ghostbusters using the Javascript selector method. I have created an event listerner to make Slimer disappear when the mouse is hovered over him. Move your mouse over the image of Slimer to see him disappear.

Playing with JavaScript

The last project I had was to move thisgs with JavaScript. In this assignement I had to make a white block (div) move from left to right using the arrow keys. I managed to do this by setting up an event listener and linking it to an action. I took this project further by making the block look like a Pac-Man ghost and enabling it to move up and down also. You can check out this project by clicking the Ghost image to the right.
